## How It Works

1. **Create Profile**
   - Pick unique username
   - Add display name & bio
   - Optional avatar image
   - Click create!

2. **Add Content**
   - Click "+ Add Content"
   - Fill in title & description
   - Paste content URL (YouTube, Medium, GitHub, etc.)
   - Add thumbnail image
   - Add tag
   - Save!

3. **Share Profile**
   - Your URL: site.com/#/yourUsername
   - Send to anyone
   - They see your portfolio
   - They can discover your content

4. **Browse Others**
   - Click "Discover"
   - Search by username
   - Filter by tags
   - Visit their profiles

## Support

2. Make sure localStorage is enabled
3. Try clearing browser cache
4. Use a modern browser (Chrome, Firefox, Safari, Edge)
