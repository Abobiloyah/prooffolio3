# ContentShow - Portfolio Showcase

A simple, beautiful proof-of-work portfolio app. Create profiles, add content links, and showcase your work.

## 🚀 Features

- **Profile Creation** - Username, display name, bio, avatar
- **Add Content** - Title, description, URL, thumbnail, tags
- **Public Profile** - Share your work at `/username`
- **Featured Content** - Highlight one item as featured
- **Discover Page** - Browse all content across users
- **Search & Filter** - Find content by username, title, or tag
- **View Tracking** - Track views on each content item
- **Zero Backend** - Everything stored in browser localStorage
- **Blue & White Theme** - Clean, minimal design
- **Mobile Responsive** - Works on all devices

## 📁 Files

- `index.html` - Complete HTML with all views and styling
- `script.js` - Full JavaScript application logic
- `README.md` - This file

## 🌐 Deployment

### Deploy to Netlify (easiest)
1. Go to https://app.netlify.com/drop
2. Drag & drop `index.html` and `script.js` files
3. Done! Your app is live

### Deploy to Vercel
1. Go to https://vercel.com/new
2. Upload the files or connect a Git repo
3. Done!

### Deploy to GitHub Pages
1. Create a GitHub repo
2. Upload `index.html` and `script.js`
3. Enable GitHub Pages in repo settings
4. Your site is live at `username.github.io/repo-name`

### Self-hosted
- Upload files to any web server
- No special setup needed

## 💡 How to Use

### Create a Profile
1. Visit the site
2. Enter username, name, bio, and optional avatar URL
3. Click "Create Profile"

### Add Content
1. Go to Dashboard
2. Click "+ Add Content"
3. Fill in:
   - **Title** - Your content title
   - **Description** - Brief description
   - **Content URL** - Link to your video, article, project, etc.
   - **Thumbnail** - Image URL (optional)
   - **Tag** - Category like "Video", "Article", "Design"
4. Click "Save"

### Share Your Profile
- Your public URL: `https://yoursite.com/#/yourUsername`
- Share this link anywhere!

### Discover Other Content
- Click "Discover" in navigation
- Search by username, title, or tag
- Filter by tag categories
- Click any item to visit creator's profile

### Feature Content
- In Dashboard, click ⭐ to mark one item as featured
- Featured item appears prominently on your profile
